# creative-eye

Teaches “design taste”; includes a 7-day training curriculum + brand-guardian pattern + cron template.

## Triggers
- "creative eye"
- "design taste"
- "brand guardian"
- "design training"
- "visual literacy"
- "aesthetic development"

## Description
A skill focused on developing design taste and visual literacy. Includes a structured 7-day training curriculum, brand-guardian patterns for maintaining design consistency, and cron templates for regular practice and reinforcement.

## Features
- 7-day design taste training curriculum
- Brand-guardian pattern implementation
- Cron templates for ongoing practice
- Visual literacy exercises
- Aesthetic development tracking

## Usage
Use when you want to improve your design sensibilities, train your eye for aesthetics, or maintain brand consistency through structured practice.

## Example
```
/creative-eye start 7-day design training
/creative-eye apply brand-guardian to this mockup
/creative-eye schedule daily design practice
```
