# Obsidian OSINT Copilot

Obsidian plugin that turns a vault into a local-first investigation workspace: entity notes (Markdown) + typed relationships (FollowTheMoney-style) plus built-in graph, timeline, and map views. AI features (entity extraction, vault Q&A/search, orchestration/planning) run via Claude Code CLI on your machine (no hosted vendor pipeline / no license key required), with prompts/rules overrideable from within the vault. Includes local geocoding via OpenStreetMap Nominatim and a YAML mirror of graph entities/connections for tooling.

## Triggers
- "obsidian osint"
- "local investigation workspace"
- "follow the money notes"
- "entity relationship mapping"
- "osint vault plugin"
- "claude code cli investigation"

## Description
An Obsidian plugin that transforms your knowledge vault into a powerful local-first OSINT investigation workspace. It combines structured note-taking with relationship mapping, visualization tools, and AI-powered features that run locally on your machine via Claude Code CLI.

## Features
- Entity notes in Markdown format
- Typed relationships using FollowTheMoney-style conventions
- Built-in graph visualization
- Timeline views for chronological analysis
- Map views for geospatial intelligence
- AI features via local Claude Code CLI:
  - Entity extraction from text
  - Vault Q&A and search
  - Investigation orchestration and planning
- No hosted vendor pipeline or license keys required
- Prompts and rules overrideable from within the vault
- Local geocoding via OpenStreetMap Nominatim
- YAML mirror of graph entities/connections for external tooling

## Usage
Use when you want to create a local, private OSINT investigation workspace within Obsidian that leverages AI capabilities without exposing your data to external services.

## Example
```
/obsidian-osint-copilot create entity note for company-xyz
/obsidian-osint-copilot map relationships between people and organizations
/obsidian-osint-copilot extract entities from leaked documents
/obsidian-osint-copilot generate timeline of events
/obsidian-osint-copilot show map of geographic connections
/obsidian-osint-copilot run investigation plan on target network
```
