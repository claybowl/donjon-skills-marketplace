# GeoIntel

Photo → GPS coordinates + confidence + Google Maps link (Gemini-based).

## Triggers
- "geointel"
- "photo to gps"
- "geolocate photograph"
- "image coordinates"
- "gemini based"
- "gps extraction"

## Description
GeoIntel is a skill that extracts GPS coordinates from photographs using Gemini-based AI. It provides not only the coordinates but also confidence scores and direct Google Maps links for easy verification and navigation.

## Features
- Photo to GPS coordinate conversion
- Gemini-based AI processing
- Confidence scoring on location estimates
- Direct Google Maps link generation
- Support for various image formats
- Batch processing capabilities
- Privacy-conscious operation

## Usage
Use when you need to determine where a photograph was taken and get actionable location data.

## Example
```
/geointel extract coordinates from vacation-photo.jpg
/geointel batch process images in folder
/geointel show confidence for location estimate
/geointel open google maps for photo location
```
