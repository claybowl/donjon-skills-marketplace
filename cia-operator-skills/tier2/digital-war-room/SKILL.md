# Digital War Room

AI-native OSINT system that runs 11+ specialized agents (GEOINT/SIGINT/SOCMINT/FININT/CYBER/ENERGY/DIPLO/etc.) under a single “Claude Sonnet supervisor.” Emphasis is on structured, inspectable outputs—escalation scores, theater context, BLUF-style briefs, and traceability back to source directories—rather than link dumps. Claims it moved from a full LangGraph design to a hybrid (rule-based dispatch + one supervisor) to reduce token cost while keeping contradiction resolution and source weighting.

## Triggers
- "digital war room"
- "osint system"
- "claude sonnet supervisor"
- "geoint sigint socmint"
- "structured osint output"
- "bluf style briefs"

## Description
An AI-native OSINT system that deploys multiple specialized intelligence agents under a central supervisor. The system emphasizes structured, inspectable outputs with clear traceability to sources, moving away from simple link dumps toward actionable intelligence with proper attribution and confidence scoring.

## Features
- 11+ specialized agents: GEOINT, SIGINT, SOCMINT, FININT, CYBER, ENERGY, DIPLO, etc.
- Single Claude Sonnet supervisor agent
- Structured, inspectable outputs
- Escalation scoring and theater context
- BLUF-style (Bottom Line Up Front) briefs
- Full traceability back to source directories
- Hybrid architecture: rule-based dispatch + supervisor
- Reduced token cost while maintaining quality
- Contradiction resolution capabilities
- Source weighting and credibility assessment
- Actionable intelligence focus

## Usage
Use when you need a comprehensive OSINT system that produces structured, actionable intelligence with clear sourcing and minimal token overhead.

## Example
```
/digital-war-room activate geoint agents for region analysis
/digital-war-room run cyber threat assessment on financial sector
/digital-war-room generate bluf brief on energy market developments
/digital-war-room show escalation scores for indo-pacific theater
/digital-war-room trace intelligence back to original sources
```
