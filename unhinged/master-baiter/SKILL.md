# master-baiter

Scam-baiting skill/project packaged as an OpenClaw skill with scripts + references + a web dashboard (FastAPI + vanilla JS). The accessible README content emphasizes repo structure and intent/dedication (Kitboga) more than mechanics; for the detailed feature set (personas, classification, reporting), you likely need to read CONTRIBUTING and the referenced skill/reference files in-repo.

## Triggers
- "master baiter"
- "scam baiting"
- "kitboga style"
- "scammer interaction"
- "fraud prevention"
- "openclaw skill"

## Description
Master-baiter is a scam-baiting skill designed to waste scammers' time and gather intelligence on fraud operations. Inspired by Kitboga's approach, it provides tools for engaging with scammers safely while collecting valuable information about their operations and techniques.

## Features
- Scam baiting capabilities
- Scripts for scammer interaction
- Reference materials for common scams
- Web dashboard (FastAPI + vanilla JS) for monitoring
- Persona development and management
- Scammer classification systems
- Intelligence reporting and reporting
- Dedication to fraud prevention and awareness

## Usage
Use when you want to engage with scammers in a controlled, safe manner to waste their time and gather intelligence on fraud operations.

## Example
```
/master-baiter start scam baiting session
/master-baiter develop persona for tech support scam
/master-baiter log interaction with irs scammer
/master-baiter generate report on gift card scam tactics
/master-baiter launch web dashboard for monitoring
```
