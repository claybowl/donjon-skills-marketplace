# Dr. Mia’s AI Agentic Skills

Huge, multi-domain skills + workflow repo (claims 676 skills across 46 categories) spanning biomedical/clinical research and also general provider/framework operations. The Skills/ tree covers areas like clinical workflows, genomics, drug discovery, software engineering, research tools/data analysis tutorials, and “agentic AI” stack curation; it also includes platform/orchestration code under platform/ plus supporting code under src/, with strategy docs and saved “sources/” snapshots when curation depends on current upstream state.

## Description
Dr. Mia's AI Agentic Skills is a comprehensive collection of AI skills and workflows spanning multiple domains with a focus on biomedical and clinical applications. The repository includes 676 skills across 46 categories, covering everything from clinical workflows to genomics and drug discovery, along with the underlying platform and orchestration code needed to deploy and manage these skills.

## Features
- 676 skills across 46 categories
- Biomedical/clinical research focus
- Clinical workflow automation
- Genomics analysis skills
- Drug discovery assistance
- Software engineering aids
- Research tools and data analysis tutorials
- Agentic AI stack curation
- Platform/orchestration code (platform/)
- Supporting code (src/)
- Strategy documentation
- Sources snapshots for reproducibility

Use when you need AI assistance in biomedical, clinical, or related research domains, or when you want to deploy a comprehensive agentic AI system for specialized workflows.
